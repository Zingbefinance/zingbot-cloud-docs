# 🚀 ZingBot Cloud MVP v0.1.0

## Objectif

Lancer une première version simple, stable et utile pour les projets Web3.

---

# Fonctionnalités

## 1. Authentification

- Inscription
- Connexion
- Déconnexion
- Gestion du profil

---

## 2. Workspace

- Créer un workspace
- Modifier un workspace
- Supprimer un workspace

---

## 3. Telegram

- Connecter un bot Telegram
- Connecter un canal
- Connecter un groupe

---

## 4. Automatisations

- Message de bienvenue
- Publication programmée

---

## 5. Dashboard

- Nombre de workspaces
- Bots connectés
- Automatisations actives
- Activité récente

---

# Hors MVP

Ces fonctionnalités seront développées après la version 0.1.0 :

- AI Assistant
- Discord
- X (Twitter)
- Token Center avancé
- Marketplace
- Trading Automation
- Application mobile

---

# Critère de réussite

Un utilisateur peut créer un compte, connecter un bot Telegram et gérer sa communauté depuis ZingBot Cloud.
